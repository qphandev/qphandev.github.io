AJAX Robot

Experimental AJAX and CSS Canvas create a robot project

Summary

A silly little project using Materialize, AJAX, PHP, and CSS Canvas. User can "make" their own robots appearance by filling in the fields.

I wanted to make something interactive and visualize OOP. It's buggy, but I'm definitely considering improving this in the future.

Input shapes and colors of the robots part. You can even name it! Silly but fun.
Designed and coded by Quynh with Bootstrap and SCSS.
