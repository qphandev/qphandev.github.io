Dice Game with React

A learning project for React

Summary

This is a learning project for me to get more used to React. I had a lot of fun with it!

The rules are:

  • Try to get all matching rolls to win(e.g. all die must be the same number face)
  • You only have three rolls
  • You can lock any die so that its value doesn't change on your next roll (of course you can lock all of it and make me look like a goof for allowing that)

It drove me kind of nuts when components didn't render after their props were updated along with all sorts of bugs in my code. My main struggle was merging my logic with its syntax and behavior that I knew little about.

Somehow it's in one piece now. Struggling with very tiny--yet very time consuming problems helped me be more familiar with what I can and cannot do in React.

Here's the source code for it on GitHub if you're interested to see: https://github.com/qphandev/dice_game

Reflection
